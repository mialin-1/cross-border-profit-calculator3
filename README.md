## 📊 计算公式

### 核心计算公式
1. **关税** = 申报成本 × 关税比例
2. **进口增值税** = (申报成本 + 关税) × 增值税率
3. **平台抽成费用** = 平台售价 × 平台抽成费率
4. **平台放款** = 平台售价 - 平台抽成费用
5. **VTB税** = 回款金额 × VTB税率
6. **国内税** = 回款金额 × 国内税率
7. **回款金额** = 平台放款 - VTB税 - 国内税 - 其他税费
8. **毛利润** = 回款金额 - 采购成本 - 各项税费
9. **毛利率** = 毛利润 ÷ 平台售价 × 100%

### 迭代计算逻辑
由于VTB税和国内税依赖于回款金额，系统实现了迭代计算方法，确保计算准确性。
